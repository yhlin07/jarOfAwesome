# Quick Start Guide üöÄ

Get your Jar of Awesome bot running in 5 minutes!

## Step 1: Get Your Tokens (2 minutes)

### Telegram Bot Token
1. Open Telegram
2. Search for `@BotFather`
3. Send `/newbot`
4. Follow instructions
5. **Copy the bot token** (looks like: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Your Telegram User ID
1. Search for `@userinfobot` in Telegram
2. Send `/start`
3. **Copy your user ID** (looks like: `123456789`)

### Anthropic API Key
1. Go to [console.anthropic.com](https://console.anthropic.com/)
2. Sign up / Log in
3. Go to API Keys
4. Create new key
5. **Copy the API key** (starts with `sk-ant-`)

## Step 2: Setup (3 minutes)

### Option A: Docker (Recommended)

```bash
# 1. Clone and enter directory
git clone https://github.com/yourusername/jar-of-awesome.git
cd jar-of-awesome

# 2. Create .env file
cp .env.example .env

# 3. Edit .env with your tokens
nano .env
# (or use any text editor)

# 4. Start the bot!
docker-compose up -d

# 5. Check it's running
docker-compose logs -f
```

**Look for:** `üöÄ Starting bot... Press Ctrl+C to stop`

### Option B: Local Python

```bash
# 1. Clone and enter directory
git clone https://github.com/yourusername/jar-of-awesome.git
cd jar-of-awesome

# 2. Create virtual environment
python -m venv venv

# 3. Activate it
source venv/bin/activate  # Mac/Linux
# OR
venv\Scripts\activate  # Windows

# 4. Install dependencies
pip install -r requirements.txt

# 5. Create .env file
cp .env.example .env

# 6. Edit .env with your tokens
nano .env  # or use any text editor

# 7. Run!
python -m src.main
```

## Step 4: Customize (Optional)

### Change Schedule Times

Edit `.env`:
```env
SCHEDULE_TIMES=07:00,13:00,17:00,21:00
```

### Change Timezone


## Troubleshooting

### "Invalid bot token"
- Check your token in `.env`
- Make sure there are no spaces
- Token should start with numbers followed by `:`

### "File not found: milestones.md"
- Make sure you're in the project directory
- Check the file exists: `ls milestones.md`

### "Missing required environment variables"
- Check `.env` file exists
- Make sure all three keys are set:
  - `TELEGRAM_BOT_TOKEN`
  - `TELEGRAM_USER_ID`
  - `ANTHROPIC_API_KEY`

### Bot not responding
- Check bot is running: `docker-compose ps`
- Check logs: `docker-compose logs -f`
- Make sure you're messaging the correct bot

### Test command
Try this in your terminal to test Telegram token:
```bash
curl https://api.telegram.org/bot<YOUR_TOKEN>/getMe
```

Should return your bot info in JSON.
